# 结果文件说明

## 2. 核心必留文件

### 2.1 `comprehensive_evaluation.csv`

**文件类型**：最终综合评估结果（最重要）

**字段说明**：

| 字段             | 含义                                       |
| ---------------- | ------------------------------------------ |
| Country          | 国家名称                                   |
| Score            | PCA 加权后的原始综合得分 $S_i$             |
| Score_Normalized | 样本内 Min–Max 归一化得分 $S'_i \in [0,1]$ |
| Rank             | 按 $S'_i$ 降序得到的排名                   |

**数学来源**：

$$
S_i = \sum_{k=1}^{m} w_k PC_{ik}, \quad
S'*i = \frac{S_i - S*{\min}}{S_{\max} - S_{\min}}
$$

**表示了什么**：

* 各国在**当前样本集合内**的相对 AI 发展能力水平
* 不表示绝对国力，只表示**相对排序与相对差距**

**用途**：

* 论文结果表（如“表5 综合评估结果”）
* 综合得分柱状图 / 排名图

**结论**：

### 2.2 `factor_importance.csv`

**文件类型**：指标（要素）重要性评估结果

**字段说明**：

| 字段           | 含义                      |
| -------------- | ------------------------- |
| Indicator      | 指标名称                  |
| Importance     | 要素重要性得分 $I_j$      |
| Normalized     | 归一化重要性 $I'_j$       |
| Contribution_% | 相对贡献度（最大值=100%） |

**数学来源**：

$$
I_j = \sum_{k=1}^{m} l_{jk}^2 \eta_k
$$

**表示了什么**：

* 在 PCA 提取的综合因子中，各指标对总体方差结构的贡献强弱
* 是“**关键驱动要素识别**”的定量依据

**用途**：

* Top-k 关键要素列表
* 要素重要性排序图（水平条形图）
* 机制分析中“谁更重要”的量化支撑

**结论**：

---

### 2.3 `pca_variance.csv`

**文件类型**：PCA 方差解释结果

**字段说明**：

| 字段             | 含义                             |
| ---------------- | -------------------------------- |
| PC               | 主成分编号                       |
| Eigenvalue       | 特征值 $\lambda_k$               |
| Variance_Ratio   | 方差贡献率 $\eta_k$              |
| Cumulative_Ratio | 累积贡献率 $\sum_{i=1}^k \eta_i$ |

**表示了什么**：

* 每个主成分在解释总体信息中的重要程度
* 决定“选几个主成分”

**用途**：

* 决定 $m$ 的依据（如 $\eta_{cum}\ge 0.85$）
* 主成分贡献率图（柱状+折线）

**结论**：

## 3. 结构分析类文件（强烈建议保留）

### 3.1 `correlation_matrix.csv`

**文件类型**：指标之间的 Pearson 相关矩阵

**字段结构**：

* 行：指标
* 列：指标
* 值：$r_{jk}$

**表示了什么**：

* 指标两两之间的线性相关程度

**用途**：

* 相关性热力图（Fig1）
* 论文中“要素高度联动”的定量证据

**结论**：

---

### 3.2 `strong_correlations.csv`

**文件类型**：强相关指标对列表

**字段说明**：

| 字段        | 含义         |
| ----------- | ------------ |
| Indicator_1 | 指标 A       |
| Indicator_2 | 指标 B       |
| Correlation | 相关系数 $r$ |

**筛选规则**：

$$|r| > 0.7$$

**表示了什么**：

* AI 发展体系中“联动最强”的要素关系

**用途**：

* 强相关对数量统计
* 弦图 / 网络图
* 机制分析（研发→人才→创新等）

**结论**：

---

### 3.3 `cluster_linkage.csv`

**文件类型**：层次聚类的 linkage 矩阵

**表示了什么**：

* 指标是如何一步步被合并为簇的
* 反映指标之间的“结构距离”

**用途**：

* dendrogram
* 聚类热力图的排序依据

### 4.2 `pca_loadings.csv`

**文件类型**：PCA 载荷矩阵

**表示了什么**：

* 每个指标在各主成分方向上的投影权重

**用途**：

* PCA 载荷热图
* PCA biplot
* 定性解释“PC1 是什么含义”
